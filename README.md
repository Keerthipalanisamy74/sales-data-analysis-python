# Sales Data Analysis using Python

This project analyzes sales data using Python and generates simple business insights.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Project Features
- Load and clean sales dataset
- Analyze sales by region
- Generate simple visualizations

## Dataset
The dataset contains sample sales information including:
- Region
- Product
- Sales amount

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the script
python sales_analysis.py
